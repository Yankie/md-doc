## Smart arrows

|  Code  | Render | Result |
| ------ | ------ | ------ |
| `-->`  | -->    | →      |
| `<--`  | <--    | ←      |
| `<-->` | <-->   | ↔      |
| `==>`  | ==>    | ⇒      |
| `<==`  | <==    | ⇐      |
| `<==>` | <==>   | ⇔      |
